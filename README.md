# NER-with-NLTK
 
 Named Entity Recognition Algorithm
 
 Identifies people, location and organisations within a piece of text
 
 This specific script compares the accuracy between manual human accuracy vs algorithm accuracy
 
 The tool kit used was Natural Language Tool Kit for extraction of entities. 
 
 The script was written using python.
 
 Programs like this sort can be used to analyse large pieces of text to extract important information for decision making
